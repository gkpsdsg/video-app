## 4. 功能模块（按优先级）

### 模块一：视频上传及存储 ⏳
**功能**：客户端选视频，分片上传，后端存 MinIO，触发转码

| 端 | 依赖 | 状态 |
|---|---|---|
| 后端 | `multer`（已装），`minio`（已装），`@nestjs/bull` + `bull`（已装），`uuid`（已装） | 待实现控制器和服务 |
| 前端 | `image_picker` / `wechat_assets_picker`，`dio` | ⏳ 待加入 pubspec.yaml |
| 额外 | FFmpeg（转码 Worker） | ⏳ 待安装 |

### 模块二：简单推荐和分发 ⏳
**功能**：首页 Feed，热度+时间衰减排序，标签加权

- Redis ZSET 缓存热门视频（`redis` 包已装）
- `@nestjs/bull` 定时更新热度（已装）

### 模块三：基础互动体系 ⏳
**功能**：点赞、评论、关注/取消

- 纯数据库操作，无特殊库需求
- Redis 可选用于点赞计数

### 模块四：用户主页 ⏳
**功能**：用户信息、视频列表、粉丝关注数

- TypeORM 关联查询 + 聚合统计

### 模块五：基础搜索 ⏳
**功能**：关键词搜索视频

- PostgreSQL 全文搜索（tsvector + 索引）
- 后期可迁至 Elasticsearch

### 模块六：简洁版播放界面 ⏳
**功能**：上下滑切换，全屏播放，预加载

| 端 | 依赖 | 状态 |
|---|---|---|
| 前端 | `video_player` + `chewie`，`cached_network_image` | ⏳ 待加入 pubspec.yaml |
| 后端 | MinIO 签名 URL 复用模块一 | 待实现 |

### 模块七：简洁版账号体系 ✅（依赖已装，逻辑待实现）
**功能**：注册、登录、JWT 鉴权、Token 刷新

| 端 | 依赖 | 状态 |
|---|---|---|
| 后端 | `@nestjs/passport` `passport` `passport-jwt` `@nestjs/jwt` `bcryptjs` | ✅ 已装，待实现控制器 |
| 后端 | `class-validator` `class-transformer`（DTO 校验） | ✅ 已装 |
| 前端 | `dio` / `http`，`shared_preferences` / `flutter_secure_storage` | ⏳ 待加入 |

### 模块八：基础内容审核 ⏳
**功能**：上传后机审 → 人工审核队列 → 管理后台

- 阿里云/腾讯云内容安全 API（HTTP 调用，无需专用 SDK）
- `@nestjs/bull` 异步审核队列（已装）
- 管理后台可先用 Swagger 文档
